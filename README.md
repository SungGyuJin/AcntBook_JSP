# AcntBook_JSP

프로그램 제작 배경
> JSP Model 1을 이용해 가장 기초적인 게시판을 구현한 후에 게시판이 아닌 복습차원에서 가계부를 한 번 만들어 보고 싶었습니다. 실제로 제가 휴대폰을 이용해서 가계부를 이용중이고, CRUD를 이용한 복습이기 때문에 응용하기 꽤 괜찮은 주제라고 생각했습니다.

* 주요 기능
1. 거래내역 등록

* 수입 or 지출 내역을 등록하는 기능 _(거래내역에 따른 카테고리 분류 버튼도 추가 예정)_

2. 거래내역 조회
* 달력으로 한눈에 볼 수 있도록 설계
